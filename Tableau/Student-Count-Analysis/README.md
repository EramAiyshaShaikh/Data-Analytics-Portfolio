## 📌 Project Overview

This project is a **Student Count Analysis Dashboard** created using Tableau.

The dashboard explores student counts in relation to different factors, including:

- Academic Pressure
- Financial Stress
- Study Satisfaction
- Sleep Duration
- Study Hours

The visualizations provide a way to explore how different academic, financial, lifestyle, and study-related factors are associated with student counts.

---

## 🎯 Project Objectives

The main objectives of this project are to:

- Analyze student counts across different levels of academic pressure.
- Explore student counts in relation to financial stress.
- Analyze the relationship between study satisfaction and student count.
- Examine student counts across different sleep-duration groups.
- Analyze student counts by study hours.

## 📊 Dashboard Analysis

### 📚 Academic Pressure vs Student Count

The dashboard visualizes student count across different academic pressure categories.

This analysis helps explore the distribution of students across varying levels of academic pressure.

---

### 💰 Financial Stress vs Student Count

The dashboard examines student counts across financial stress categories.

This allows users to explore the distribution of students experiencing different levels of financial stress.

---

### 😊 Study Satisfaction vs Student Count

The visualization analyzes student counts across study satisfaction levels.

This provides an overview of how students are distributed across different satisfaction categories.

---

### 😴 Sleep Duration vs Student Count

The dashboard compares student counts across sleep-duration groups, including:

- Less than 5 hours
- 5–6 hours
- 7–8 hours
- More than 8 hours

This helps explore the distribution of students based on their reported sleep duration.

---

### 📖 Study Hours vs Student Count

The dashboard visualizes student counts across different study-hour categories.

This provides an overview of how students are distributed based on their study time.

---

## 💡 Key Observations

The dashboard allows users to explore:

- Differences in student counts across academic pressure levels.
- Student distribution across financial stress categories.
- Study satisfaction patterns.
- Student distribution by sleep duration.
- Differences in student counts across study-hour groups.

> Note: The dashboard provides visual comparisons between these factors and student count. Further statistical analysis would be required to establish causal relationships.
